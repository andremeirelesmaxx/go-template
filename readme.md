# Go App Template

This template runs go on docker, you not need to have go installed on machine

## Uses 

* Air - for hot reload on development
* fx - for app runner and dependency injection
* cobra - as cli maker

## How use

Clone or donwload the repository, change project name on `go.mod` for your app name and set your reposiutory with `git remote add orogin [YOUR_URL]` or `git remote set-url origin [YOUR_URL]`.

Replace the `github/andresmeireles/go-template` for your project name.
